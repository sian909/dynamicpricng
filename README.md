# 1인 1가격 시뮬레이션 — 안정형 한 페이지 버전

## 화면 구성
- 사용자 3명
- 실시간 데이터베이스 및 그래프
- 한 페이지 4분할 구조

## 구매 완료 동작
- 각 사용자 패널의 `구매 완료`가 마지막 이벤트로 DB에 저장됩니다.
- 구매 완료한 사용자만 체류시간과 클릭 수집이 종료됩니다.
- 다른 두 사용자는 계속 시연할 수 있습니다.
- 서버에서도 종료된 사용자에 대한 추가 이벤트 저장을 차단합니다.

## 실행
```bash
python3 -m pip install -r requirements.txt
python3 app.py
```

접속:
http://127.0.0.1:5001
